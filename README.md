# ReUseX

A trusted marketplace for second-hand buying, selling, and peer-to-peer rentals.

## Vision

To make second-hand commerce and rentals more accessible, affordable, and trustworthy.

## MVP Features

- User Authentication
- Product Listings
- Buy & Sell Marketplace
- Peer-to-Peer Rentals
- User Profiles
- Ratings & Reviews
- Messaging System
- Admin Dashboard

## Tech Stack

### Frontend
- Next.js
- TypeScript
- Tailwind CSS

### Backend
- Next.js API Routes

### Database
- PostgreSQL
- Prisma ORM

### Authentication
- Clerk

### Storage
- Cloudinary

### Deployment
- Vercel
- Neon PostgreSQL
